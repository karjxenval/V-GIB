# Repository description

Official reproducibility code for **V-GIB (Variational Geometric Information Bottleneck)**, a framework for geometry-aware representation learning under label scarcity. The code trains matched ERM, VIB, V-GIB, and ablated V-GIB variants; computes predictive metrics, curvature proxies, intrinsic-dimension diagnostics, and utility proxies; and generates CSV summaries and figures for the accompanying manuscript.

Suggested GitHub short description:

> Reproducibility code for V-GIB: geometry-aware information bottleneck learning for data-scarce representation learning.
